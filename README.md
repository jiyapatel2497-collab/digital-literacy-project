Project Overview
This project focuses on building essential digital literacy skills, including awareness, professional online presence, use of digital platforms, communication etiquette, and cyber safety. It is divided into five modules, each covering a key aspect of digital literacy.

Modules Covered
Module 1: Digital Literacy Awareness
Created an infographic explaining digital literacy, useful tools, safe internet practices, and email etiquette.

Module 2: Digital Portfolio
Set up professional profiles on GitHub, LinkedIn, and Stack Overflow to build an online presence.

Module 3: Platforms & Collaboration
Practiced coding on HackerRank and created a Digital Literacy Quiz using Google Forms.

Module 4: Email Etiquette
Drafted professional emails and created a social media do’s and don’ts checklist.

Module 5: Cybercrime Awareness
Studied phishing attacks and created a prevention checklist for staying safe online.
